Predict::Predict ( )

The Predict class is used to train a logistic regression algorithm using a specific lights state data to predict future scheduling state events.

If the algorithm does not perform at a high enough level of certainty, than the Predict class will opt to use a most-frequent algorithm to decide scheduling.
